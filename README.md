# Bill-split-bot

Телеграм-бот для расчёта совместных затрат в поездках

# Описание
В групповых путешествиях часто возникает ситуация, когда один человек оплачивает услуги за всех, а потом сложно вспомнить кто кому и сколько должен. Ручной учет в заметках или таблицах неудобен и подвержен ошибкам.

Бот предназначен для управления совместными расходами в путешествиях с друзьями. Решает проблему учета общих трат и расчета долгов между участниками поездки.

# Конфигурация

Для работы нужно добавить файл config.py с переменной:
```
BOT_TOKEN=<токен бота>
```

# Ключевые особенности

1. Умный учет всех трат
- Учитываются все расходы независимо от того, кто именно платил
- Поддержка разных валют (доллары, рубли и др.)
- Категоризация расходов для удобного анализа

2. Распознавание чеков по фото
- Просто загружаете фото чека - бот автоматически распознает сумму и детали
- Не нужно вручную вводить данные
- Работает с российскими чеками в рублях

3. Гибкая система участников
- В разных платежах может участвовать разный набор людей
- Динамическое подключение/отключение участников
- Пропорциональное распределение затрат

4. Оптимизированные переводы
- Автоматический расчет минимального количества переводов
- Учет взаимных долгов "по цепочке"
- Интеграция с банковскими реквизитами участников
 
5. Персональная статистика
- Анализ личных расходов по категориям
- Статистика именно ваших трат в поездке
